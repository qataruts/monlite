👋 Everything here runs 100% in your browser on SQLite-WASM — no server, no backend. The data lives in one .db file in this tab. Each tab is a different monlite package — every query shows live in the SQL log →

@monlite/core MongoDB-style document collections + typed queries + reactive watch(). This grid is live — add or delete and it updates instantly.

@monlite/queue Durable job queue — jobs persist in the DB, a worker claims & runs them, with retries & backoff (~30% of demo jobs fail once to show a retry).
